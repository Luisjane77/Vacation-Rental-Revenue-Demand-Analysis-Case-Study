# Segmentation Analysis

This section focuses on segmentation and behavioral pattern analysis in the vacation rental market.

Its purpose is to move beyond aggregate indicators and explore whether different customer or rental profiles can be identified through clustering-based analysis. This perspective is useful because it helps reveal patterns that may remain hidden when the market is analyzed only at a general level.

The segmentation work included in this project is intended to support stronger business interpretation by identifying meaningful differences across profiles, behaviors, or rental patterns. In a vacation rental context, this type of analysis can be valuable for customer understanding, commercial positioning, and more targeted decision-making.

This section is closely connected to the dashboard outputs included in the repository.

## Main Focus Areas

- Behavioral clustering
- Profile-based interpretation
- Segmentation-oriented business insights
- Visual support for customer and rental pattern analysis
